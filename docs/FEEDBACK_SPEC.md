# מפרט משובים לתלמידים

## עקרונות מנחים

- **ללא ציונים מספריים** - לא להציג מספרים לתלמיד
- **ללא סף עבירה** - לא לציין כמה צריך
- **ללא שפת מבחנים** - לא "עברת/נכשלת"
- **לשון רבים תמיד** - "נסו", "המשיכו", "כל הכבוד"
- **צבעים ניטרליים** - כחול לציון נמוך, ירוק לגבוה
- **מינימליזם** - פחות טקסט, יותר ברור

---

## הודעות משוב לפי ציון

| טווח ציון | הודעה |
|-----------|-------|
| 0-30 | `נסו שוב` |
| 31-50 | `נסו שוב` |

| 61-79 | `יפה` |

## עיצוב ויזואלי

### צבעים

| מצב | צבע |
|-----|-----|
| ציון נמוך (0-60) | כחול רך (`sky-100`, `sky-200`) |
| ציון גבוה (61+) | ירוק (`emerald-100`, `green-500`) |

### אייקונים

| מצב | אייקון |
|-----|--------|
| ציון נמוך (רגיל) | ללא אייקון מיוחד |
| ציון נמוך + שאלה מותאמת | אייקון חזרה 🔄 |
| ציון גבוה | וי ירוק ✓ |

| מצב | טקסט כפתור |
|-----|------------|
