# System Manifesto (The Rules)

## A. General Navigation & UI
- **Header**: Must always show "User Name", "Points Balance", and "Notification Bell".
- **Navigation**: 
  - "Next" button must be disabled until an answer is selected. 
  - "Back" button allowed only in practice mode.
- **Colors**: 
  - Primary action buttons must be Blue (#0056b3).
  - Destructive actions Red.

## B. Pedagogical Features
- **Scoring**: 
  - Standard question = 10 points. 
  - Bonus question = 20 points.
- **Hints**: 
  - Every question MUST have a "Lightbulb" icon for a hint. 
  - Clicking it deducts 2 points.
- **Feedback**: 
  - Immediate feedback is required. 
  - Green V for correct.
  - Red X for wrong + textual explanation.

## C. Gamification Modules
- **Streak**: Answering 3 correct in a row must trigger a "Fire" animation.
- **Badges**: Finishing a unit with >90 score awards a "Gold Medal".
