# 🚀 מדריך מהיר - יצירת אינפוגרפיקה

## צעד 1: פתח Teacher Cockpit
```
נווט ליחידת לימוד → לחץ על "מערך שיעור למורה" או "ערוך"
```

## צעד 2: בחר בלוק
```
Hover על כל בלוק תוכן (טקסט, שאלה, וכו')
יופיעו בקרות בפינה השמאלית העליונה
```

## צעד 3: לחץ על כפתור האינפוגרפיקה 📊
```
כפתור סגול עם אייקון תרשים
יפתח תפריט עם 4 אפשרויות
```

## צעד 4: בחר סוג
| אייקון | סוג | מתי להשתמש |
|--------|-----|-----------|
| 📋 | תרשים זרימה | תהליכים, שלבים |
| ⏱️ | ציר זמן | אירועים היסטוריים |
| ⚖️ | השוואה | ניגודים, טבלאות |
| 🔁 | מחזור | לולאות, מחזורים |

## צעד 5: המתן
```
⏳ "יוצר אינפוגרפיקה..." - 10-20 שניות
✅ התמונה תופיע אוטומטית אחרי הבלוק
```

---

## 💡 טיפ מהיר
**הטקסט הכי טוב לאינפוגרפיקה:**
- ✅ מאורגן בנקודות / רשימה
- ✅ יש רצף ברור (1, 2, 3...)
- ✅ מכיל מושגי מפתח בולטים
- ❌ פסקה רציפה ארוכה ללא מבנה

---

## 🎨 דוגמה
**טקסט:**
```
מחזור המים כולל 4 שלבים:
1. אידוי - המים מתאדים מהים
2. התעבות - אדי המים הופכים לעננים
3. משקעים - גשם יורד מהעננים
4. נגר - המים זורמים חזרה לים
```

**בחר:** 🔁 מחזור
**תוצאה:** תרשים מעגלי עם 4 שלבים וחצים!
